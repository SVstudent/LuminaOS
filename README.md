# LuminaOS | AI-Native Education Platform

LuminaOS is a premium, AI-native education platform designed to provide a modern, highly engaging learning experience. Built with React, TypeScript, and Google GenAI, it features a Socratic AI tutor, a neural worksheet forge, and professional teacher analytics.

## Features
- **Lumina Sync**: Real-time voice + whiteboard Socratic tutoring.
- **Neural Forge**: AI-powered curriculum and worksheet generation.
- **Professional Analytics**: Deep insights into student engagement and mastery.
- **Glassmorphic UI**: High-end, modern aesthetic for a premium feel.

## Tech Stack
- React 19
- TypeScript
- Tailwind CSS
- Google GenAI (Gemini 2.5 Flash)
- Firebase Firestore & Auth
